#Da fare subito
installare node.js, ultima versione LTS

##Setup Ambiente
###estensioni
scaricate le estensioni: ESLint, prettier - code formatter, Tailwind CSS IntelliSense, Todo Tree.

Per prettier consiglio poi di fare ctrl + , (settings), poi cercare nella barra ricerce Editor: Default Formatter e selezionare prettier.
Infine cervare nella stessa barra Editor: Format On Save e spuntare (cliccare casella).
Ora quando salverete cio che avete scritto, prettier sistemera com'è scritto il codice, rendendo tutto uguale

##dipendenze

nel terminale entrare in mobile-app (cd mobile-app) e scrivere npm install
nel terminale entrare in web-app (cd web-app) e scrivere npm install
